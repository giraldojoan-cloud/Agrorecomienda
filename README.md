# Agrorecomienda
En Colombia, miles de campesinos toman decisiones de siembra basadas en intuición, perdiendo dinero si el suelo no es apto. ¿Qué pasaría si la Inteligencia Artificial les dijera exactamente qué sembrar?. Este es mi proyecto personal mostrando una alternativa a este problema.
### Ingeniería de Activos (Tech Stack)
Para alcanzar este nivel de precisión, el sistema implementa:
- **Algoritmo de Alto Rendimiento:** Uso de **Ramdom forest** para capturar patrones no lineales en Big Data agrícola.
- **Ingeniería de Ratios:** Creación de variables inteligentes como los ratios **Calcio/Magnesio** y **Fósforo/Potasio**, fundamentales para determinar la salud financiera y biológica del cultivo.
- **Contexto Geográfico:** Codificación avanzada de Departamentos y Municipios para integrar el factor territorial en la predicción.

###  Big Data Analytics
- **Volumen de Datos:** Entrenamiento sobre una muestra refinada de **49,363 registros reales** de laboratorios colombianos.
- **Optimización de Recursos:** El modelo se enfoca en los **10 cultivos líderes** de la economía nacional, asegurando una especialización técnica superior.
